Reproductor de Música con Lista Circular en C++

Este proyecto consiste en el desarrollo de un reproductor de música utilizando listas circulares dobles en C++. El programa permite agregar canciones, mostrar la playlist completa y reproducir canciones de manera dinámica utilizando las teclas de dirección del teclado.

# Funcionalidades

* Agregar canciones a la playlist.
* Mostrar todas las canciones registradas.
* Reproducir canciones en orden circular.
* Navegar entre canciones con:

  * Flecha derecha → siguiente canción.
  * Flecha izquierda → canción anterior.
  * ESC → regresar al menú principal.

# Tecnologías utilizadas

* Lenguaje C++
* Estructuras dinámicas
* Lista circular doble
* Punteros

## Objetivo

Aplicar el uso de listas enlazadas circulares para simular el funcionamiento básico de un reproductor de música, fortaleciendo conocimientos sobre memoria dinámica, estructuras de datos y recorridos circulares.
